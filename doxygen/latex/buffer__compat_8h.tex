\subsubsection[{evbuffer\_\-cb}]{\setlength{\rightskip}{0pt plus 5cm}typedef void($\ast$ {\bf evbuffer\_\-cb})(struct {\bf evbuffer} $\ast$buffer, size\_\-t old\_\-len, size\_\-t new\_\-len, void $\ast$arg)}\label{buffer__compat_8h_a613068ef7e1be05deead9c9b6f021158}


Type definition for a callback that is invoked whenever data is added or removed from an \doxyref{evbuffer}{p.}{structevbuffer}. An \doxyref{evbuffer}{p.}{structevbuffer} may have one or more callbacks set at a time. The order in which they are executed is undefined.

A callback function may add more callbacks, or remove itself from the list of callbacks, or add or remove data from the buffer. It may not remove another callback from the list.

If a callback adds or removes data from the buffer or from another buffer, this can cause a recursive invocation of your callback or other callbacks. If you ask for an infinite loop, you might just get one: watch out!

\item[{\em buffer}]the buffer whose size has changed \item[{\em old\_\-len}]the previous length of the buffer \item[{\em new\_\-len}]the current length of the buffer \item[{\em arg}]a pointer to user data \end{DoxyParams}

Replace all callbacks on an \doxyref{evbuffer}{p.}{structevbuffer} with a single new callback, or remove them. Subsequent calls to \doxyref{evbuffer\_\-setcb()}{p.}{buffer__compat_8h_a410c80ef25950aed88a08141278f23ab} replace callbacks set by previous calls. Setting the callback to NULL removes any previously set callback.

\begin{Desc}
\item[{\bf Deprecated}]This function is deprecated because it clears all previous callbacks set on the \doxyref{evbuffer}{p.}{structevbuffer}, which can cause confusing behavior if multiple parts of the code all want to add their own callbacks on a buffer. Instead, use \doxyref{evbuffer\_\-add()}{p.}{buffer_8h_a69c3b8893a49bdd919642c6920c1fcb4}, evbuffer\_\-del(), and evbuffer\_\-setflags() to manage your own \doxyref{evbuffer}{p.}{structevbuffer} callbacks without interfering with callbacks set by others.\end{Desc}

\item[{\em buffer}]the \doxyref{evbuffer}{p.}{structevbuffer} to be monitored \item[{\em cb}]the callback function to invoke when the \doxyref{evbuffer}{p.}{structevbuffer} is modified, or NULL to remove all callbacks. \item[{\em cbarg}]an argument to be provided to the callback function \end{DoxyParams}
