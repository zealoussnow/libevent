\subsection{Detailed Description}
Pointer to a position within an \doxyref{evbuffer}{p.}{structevbuffer}. Used when repeatedly searching through a buffer. Calling any function that modifies or re-\/packs the buffer contents may invalidate all evbuffer\_\-ptrs for that buffer. Do not modify these values except with evbuffer\_\-ptr\_\-set.
